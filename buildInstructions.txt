Как настраивать нашу Android Studio

Запускаете установщик, устанавливаете. Желательно папку, в которую разрешена записаь (не системная)

Запускаете студию. Открываете проект SpeechRecognitionAndroid

Далее, три полоски слева вверху -> Файл -> Настройки -> Языки и фреймворки -> Android SDK

Ставьте 14.0 и 11.0 (для более старых)

Жмите применить, ждите установки

Далее там же, в этом меню, build, execution... -> build tools -> gradle

Создайте папку, назовите как угодно (я назвал gradles) и положите её внутрь папки c SDK (папку вибирали при установке)
На самом деле не важно, куда класть эту папку, важно её прописать, чтобы знать, где лежат версии.

Ниже, где написано gradle progects в поле distribution поставить wrapper

Далее запускаете синхронизацию (три полоски -> файлы -> синк проект с градлес

Затем кнопка run сверху
